我们成立EOS Scholar测试网络的初衷是希望可以为出块节点候选者们提供一个实时在线且安全的EOS测试网环境, 同时我们也会提供相应的学习材料。

## 目标和愿景

- 拥有在EOS测试网学习的机会
- 最佳实践合集
- EOS.IO 出块节点服务器配置
- 故障排除和技术支持
- 多语言文档
- 出块节点高频率轮动
- 安全性及网络审计
- 主节点/从节点/备用节点规则和相关配置
